Google Calendar

Google Calendar Sync WIP

Sync your EduPage timetable into a dedicated Google Calendar and keep the current week updated when rooms, teachers, or lessons change. This feature is still a work in progress — expect rough edges around halfyear sync and event colors.

OAuth Required

Google Calendar writes do not work with a simple API key. Use a Google OAuth Client ID and Client Secret here instead. For a hobby setup, a free personal Google account and a free Google Cloud project are enough. You do not need Chrome Web Store publishing for the current setup.

Google Setup

Create a Google Cloud OAuth Client ID, not an API key. You only need to do this once.

1. Open Google Cloud Console and create or select a project.

2. Enable the Google Calendar API for that project.

3. Open APIs & Services → Credentials.

4. Create an OAuth client ID of type Web application.

5. Add the exact redirect URI shown below to Authorized redirect URIs.

6. Copy the Client ID and Client Secret, paste them below, then click Connect Google.

7. No payment is required for normal hobby use.

Google Credentials

Paste your Google OAuth Client ID and Client Secret here. API keys are not supported for calendar writes.

Put that redirect URI into your Google OAuth client exactly as shown. If Google says redirect_uri_mismatch, this field is the fix.

Week mode syncs the current whole week. On Saturday and Sunday it syncs the upcoming week instead. Halfyear mode repeats the detected timetable template through the current halfyear and then keeps the active week updated from EduPage.

School events are currently conservative: only explicitly marked tests and exams are added when that toggle is enabled.

Halfyear coverage defaults to future events only. Whole current halfyear is available too, but it is slower and may take noticeably longer to finish.

The second halfyear ends on June 30, so this sync will not continue into July or August.

Halfyear sync is intentionally throttled so Google is less likely to reject the extension with rate limits.
